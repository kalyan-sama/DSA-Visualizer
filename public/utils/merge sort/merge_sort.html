Merge Sort Algorithm

Basic Definition

Merge sort is a divide-and-conquer algorithm that recursively divides an array into smaller subarrays, sorts them, and then merges them back together to produce a sorted array.

How It Works

Merge sort follows these steps:

  1. Divide: The algorithm starts by dividing the unsorted array into n subarrays, each containing one element (an array of one element is considered sorted).
  2. Conquer: Recursively sort the subarrays by repeatedly dividing them until each subarray has only one element.
  3. Merge: Repeatedly merge the sorted subarrays to produce new sorted subarrays until there is only one subarray remaining. This will be the sorted array.

Detailed explanation of the merge process:

  • Compare the elements of two subarrays starting from the first element of each.
  • Select the smaller element and add it to the merged array.
  • Move to the next element in the subarray from which the element was selected.
  • Repeat this process until all elements from both subarrays are merged.
  • If one subarray is exhausted, add all remaining elements from the other subarray to the merged array.

Time and Space Complexity

  • Time Complexity: O(n log n) in all cases
  • Space Complexity: O(n)

Advantages

  • Stable sorting algorithm
  • Guaranteed O(n log n) time complexity
  • Efficient for large datasets

Applications

  • External sorting
  • Sorting linked lists
  • Inversion count problem
  • Used in various database systems
